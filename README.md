# Dipsy Paris — Soirée Festive

Site de réservation pour la soirée Dipsy Paris du 28 Mai 2026.

## Publier sur GitHub Pages

1. Créez un nouveau repository sur [github.com](https://github.com)
2. Uploadez le fichier `index.html`
3. Allez dans **Settings → Pages**
4. Sous **Source**, choisissez **Deploy from a branch**
5. Sélectionnez la branche **main** et le dossier **/ (root)**
6. Cliquez **Save**
7. Votre site sera disponible à l'adresse : `https://[votre-username].github.io/[nom-du-repo]`
